### 1.项目放到网站子目录

```nginx
#nginx 配置
location /datasource {
            alias /Users/binbin.wang/workspace/work-1/ai-doc/build/datasource;
            index index.html index.htm;
        }
```

webpack 打包设置 `PUBLIC_URL='/datasource'` ,`public_url` 指定页面静态资源加载路径

访问路径为 `http://localhost:3000/datasource`
